Seu site, sua cara

Os conceitos de HTML e CSS são fundamentais para o desenvolvimento web.

HTML (HyperText Markup Language)



Estruturação: HTML é a linguagem de marcação usada para estruturar o conteúdo da web. Ele define a estrutura básica de uma página web usando elementos como
, < header >, < footer >, < section >, entre outros.

Tags: HTML usa tags para criar elementos. Cada tag tem uma função específica, por exemplo, < h1 > para cabeçalhos, < p > para parágrafos, < a > para links, < img > para imagens, e assim por diante.

Atributos: Tags HTML podem ter atributos que fornecem informações adicionais sobre o elemento, como: id, class, src, href, entre outros.

Semântica: HTML5 introduziu elementos semânticos como: < article >, < nav >, < aside >, e < main >, que ajudam a definir a estrutura e o significado do conteúdo.

CSS (Cascading Style Sheets)



Estilização: CSS é usado para estilizar a aparência de uma página web. Ele define como os elementos HTML devem ser exibidos em termos de layout, cores, fontes, espaçamento, e muito mais.

Seletores: CSS utiliza seletores para aplicar estilos a elementos HTML. Seletores podem ser simples por exemplo: h1, .classe, #id ou complexos por exemplo: div > p, a:hover.

Propriedades e Valores: Cada regra CSS consiste em uma propriedade como: color, font-size, margin e um valor associado: red, 16px, 10px.

Cascading: O conceito de cascata em CSS determina como os estilos são aplicados e resolvidos, especialmente quando há múltiplas regras que se aplicam ao mesmo elemento.

Box Model: O modelo de caixa é um conceito fundamental que descreve como os elementos são renderizados na tela, considerando margens, bordas, preenchimento e conteúdo.

Layouts Responsivos: CSS permite criar layouts que se adaptam a diferentes tamanhos de tela usando unidades flexíveis, media queries, e frameworks como Bootstrap. Esses conceitos formam a base para criar páginas web bem estruturadas e estilizadas, proporcionando uma boa experiência de usuário.

Dicas de como estruturar



Estrutura Semântica - O que é?


As tags semânticas dp HTML ajudam a descrever o propósito do conteúdo dentro de um documento. Elas tornam o código mais legível e ajudam ferramentas de acessibilidade e motores de busca a entenderem melhor o conteúdo.

Exemplo de Código HTML



          <!DOCTYPE html>
          <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <title>Página com Estrutura Semântica</title>
          </head>
          <body>
            <header>
              <h1>Bem-vindo ao Meu Blog</h1>
            </header>
            <nav>
              <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#sobre">Sobre</a></li>
                <li><a href="#contato">Contato</a></li>
              </ul>
            </nav>
            <section>
              <article>
                <h2>Título do Artigo</h2>
                <p>Este é o conteúdo do artigo...</p>
              </article>
            </section>
            <footer>
              <p>Copyright © 2024</p>
            </footer>
          </body>
          </html>

Neste exemplo, usamos < header >, < nav >, < section >, < article >, e < footer > para criar uma estrutura clara e semântica.


Agora vamos ver em detalhes cada uma dessas tags semânticas:


< header > O Que é?


A tag < header > é usada para definir um cabeçalho para um documento ou uma seção do documento.


< nav > O Que é?


A tag < nav > é usada para definir um conjunto de links de navegação. Geralmente contém menus de navegação que levam a outras páginas ou seções do site.


< section > O Que é?


A tag < section > é usada para definir seções em um documento, como capítulos, cabeçalhos ou qualquer outro grupo de conteúdo temático. Ajuda a organizar o conteúdo em blocos temáticos, facilitando a leitura e a estruturação do documento.


< article > O Que é?


A tag < article > é usada para definir conteúdo independente e auto-contido.Ideal para artigos de blog, notícias, postagens de fórum ou qualquer outro conteúdo que possa ser distribuído de forma independente.


< footer > O Que é?


A tag < footer > é usada para definir um rodapé para um documento ou uma seção do documento. Normalmente contém informações sobre o autor, direitos autorais, links de contato, e outros dados relacionados ao documento.

Neste exemplo, criamos uma página com um layout visualmente agradável e responsivo. Você pode expandir e personalizar o design adicionando estilos, cores, fontes e outras melhorias visuais


Reset Básico: Removemos a margem e o preenchimento padrão de todos os elementos e ajustamos o box-sizing para border-box para facilitar o dimensionamento.

Body: Definimos a fonte e o line-height para melhorar a legibilidade.

Header: Estilizamos o cabeçalho com um fundo escuro e texto claro. Centralizamos o texto e adicionamos espaçamento.

Nav: Estilizamos a navegação para exibir os links horizontalmente no desktop e verticalmente em dispositivos móveis. Adicionamos um efeito hover nos links.

Section: Adicionamos espaçamento interno e uma borda inferior para separar visualmente cada seção.

Footer: Estilizamos o rodapé de forma semelhante ao cabeçalho.

Media Query: Tornamos a navegação responsiva, ajustando o layout para dispositivos com largura máxima de 600px.
